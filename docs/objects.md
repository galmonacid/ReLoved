# Modelo de Objeto (publicación)

## Campos obligatorios
- Foto del objeto
- Título
- Ubicación aproximada (barrio/zona)
- Fecha de publicación (automática)

## Campos opcionales
- Ninguno en el MVP

## Funcionalidades relacionadas
- Publicar un objeto disponible.
- Mostrar en lista de búsqueda filtrada por radio.
- Permitir que usuarios interesados contacten con el dueño.
