# Modelo de Usuario

## Campos obligatorios
- Email
- Contraseña
- Nombre visible

## Opcionales en el MVP
- Ninguno

## Funcionalidades relacionadas
- Publicar objetos.
- Buscar objetos dentro del radio definido.
- Contactar con otros usuarios mediante el formulario de email.
- Valorar otros usuarios tras completar un intercambio.
